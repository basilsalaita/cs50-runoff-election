# Runoff Election Simulator (C)

This project implements a ranked-choice runoff voting system in C.

The program records voter preferences, tabulates votes, eliminates candidates with the fewest votes, and determines the winner through multiple rounds.

## Skills
- C Programming
- Algorithms
- Data Structures
- Problem Solving

## Development Notes
This project was developed as part of Harvard's CS50 course. 
AI tools were occasionally used to assist with debugging and 
clarifying programming concepts during development.
